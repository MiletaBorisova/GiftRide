# GiftRide – Web Application for Selling Experience Vouchers

## 📌 Overview

**GiftRide** is a web-based application developed as a diploma project. The platform functions as an online store for purchasing vouchers for extreme and memorable experiences.

The goal of the project is to provide users with an intuitive and modern system for browsing, selecting, and reserving experience-based products.

---

## 🎯 Project Goals

* Develop a fully functional web application for selling vouchers
* Research existing platforms and best practices
* Design a scalable and user-friendly architecture
* Implement core e-commerce functionalities

---

## 🛠️ Technologies Used

* **Backend:** ASP.NET Core (MVC Architecture)
* **Programming Languages:** C#, HTML, CSS, JavaScript
* **Frontend Tools:** Razor, Bootstrap
* **Database:** Microsoft SQL Server (Code First approach)
* **Development Tools:** Visual Studio 2022, Rider, SSMS

---

## 🧱 Architecture

The application follows a **three-layer architecture**:

* **Presentation Layer** – User interface and views
* **Business Logic Layer** – Core application logic
* **Data Layer** – Database access and management

The project is structured using the **MVC design pattern**, ensuring separation of concerns and maintainability.

---

## 🗄️ Database

* Built using **Code First approach**
* Includes custom tables tailored to the project requirements
* Designed for scalability and easy future extension

---

## ⚙️ Core Components

* **Controllers:** Handle user requests and application flow
* **Services:** Implement business logic via interfaces
* **ViewModels:** Transfer data between views and controllers
* **Views:** User interface for interaction
